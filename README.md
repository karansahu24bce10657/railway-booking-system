                                    Railway Ticket Booking System

## Overview
A console-based railway ticket booking system built in Java that allows users to book tickets, view available trains, and manage bookings.

## Features
- Book railway tickets
- View available trains and stations
- Display all booked tickets
- Input validation and error handling

## Technologies Used
- Java
- Object-Oriented Programming
- ArrayList data structures

## Installation & Run
1. Clone repository: `git clone https://github.com/yourusername/railway-booking-system.git`
2. Compile: `javac Main.java`
3. Run: `java Main`

## Testing
Run the application and test various scenarios:
- Valid ticket booking
- Invalid station selection
- View empty ticket list
